## Ready for new create app?

Once an app is ready:
1. cd to this folder
2. do: git checkout -b app_yourbranch
3. copy the files
4. git commit -am "app_yourbranch" && git push
5. your new branch is ready for scaffolding via sergeant create yourbranch folder
